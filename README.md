# gear-up
🇧🇷 Gear Up é o primeiro projeto de um ecossistema SaaS inspirado no sistema solar dentro da Via Láctea, representando o “Sol” como o início de todo o universo de produtos.
🇺🇸 Gear Up is the first project of a SaaS ecosystem inspired by the solar system within the Milky Way, representing the “Sun” as the starting point of the entire product universe.
